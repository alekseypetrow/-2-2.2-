## Тема 2 (задание 2.2)

##  Решение задач с использованием прикладных программ, в том числе открытого программного обеспечения

**Задачи:**
1. Экономия денежных средств образовательного учреждения и самих обучающихся. 
2. Обеспечение возможности изучения работы программного обеспечения. 
3. Обучение написанию качественных программ.
4. Обучение грамотному использованию свободного программного обеспечения.
5. Независимость от поставщика.
6. Возможность свободно делиться программами.
7. Доступность значительного объёма прикладного свободного ПО.
8. Возможность глубокого изучения и доработки программ.

**Продукты:**
1. ALT- Linux, EdUbuntu, Debian,MOPSLinux	- Операционные системы (для серверов и рабочих станций)
2. OpenOffice -	Офисный пакет (текстовый редактор «Writer», редактор математических формул «Math», электронная таблица «Calc», система управления базами данных «Base», редактор презентаций «Impress»
3. Scratch (аналог Logo )	- Визуальная среда программирования
4. Lazarus ( аналог Pascal )-	Визуальная среда программирования
5. Eclipse - Визуальная среда программирования
6. NetBeans	- Визуальная среда программирования
7. Mozilla Firefox	- Web-браузер
8. Mozilla Thunderbird	- Клиент электронной почты
9. KompoZer	HTML - редактор
10. Gimp	- Редактор растровой графики
11. Inkscape	- Редактор векторной графики
12. QCad	- Профессиональная система черчения
13. Scribus	- Настольная издательская система
14. Audacity	- Редактор звуковых файлов
15. Avidemux	- Видеоредактор
16. Blender	- Редактор трехмерной графики
17. Skype	- Видеосвязь, служба мгновенных сообщений
18. MySQL	- Система управления базами данных
19. Google Apps	- Software as a Service (ПО как сервис)
20. Альт Образование	- Операционная система для образовательных учреждений
